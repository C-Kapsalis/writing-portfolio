PyMC-Marketing
--------------

- **Contribution:** *(add PR link once opened)*

PyMC-Marketing is a Bayesian marketing-analytics library (media-mix modelling,
customer lifetime value) — squarely my domain. Its docs are Sphinx-based and
the project actively welcomes documentation pull requests. I am contributing a
glossary that bridges marketing vocabulary and Bayesian terminology, so a
marketer has a hook to hang unfamiliar statistical concepts on, plus a
short explanation page aimed at that reader rather than at a statistician.

How I work on these
-------------------

I read ``CONTRIBUTING.md`` first, build the docs locally to confirm my change
renders, and claim an issue before starting. I frame each change in Diátaxis
terms in the pull-request description — for example, "this page mixed reference
and explanation, so I split them" — because the reasoning is the part worth
reviewing, not just the diff.
