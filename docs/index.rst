Christoforos Kapsalis
=====================

**Technical writer for data and developer tools.**

I document the kind of software I come from: data platforms, analytics tooling,
and the systems that sit between them. My background is in data science and
marketing analytics, so I write for the two audiences these products actually
have — the engineer who operates the system, and the analyst who depends on it.

I work docs-as-code (Git, Sphinx, reStructuredText, published from CI) and
structure documentation with `Diátaxis <https://diataxis.fr>`_. I care about
the same thing in every piece: does this serve the reader's real need at the
moment they have it — and only that need?

This site is itself written in reStructuredText, built with Sphinx, and
deployed to GitHub Pages from a GitHub Actions pipeline. How I built it is
:doc:`documented here <colophon>` as one of the samples.
